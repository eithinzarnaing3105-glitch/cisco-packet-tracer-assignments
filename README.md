# cisco-packet-tracer-assignments
Collection of Cisco Packet Tracer  assignments and network simulations from networking course covering routing, switching, VLANs, and protocols.

## Technologies Used
- Cisco Packet Tracer

## Repository Contents
Each folder contains '.pkt' files that can be opened in Cisco Packet Tracer:
- Lab assignments with configurations
- Network topology designs
- Protocol implementations
